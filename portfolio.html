Pizza Program 2.0 is a simulated program where users can create a pizza based on their personal choices.

Users can choose to spend which sauce and which toppings they would like to include on their pizza.

Once the toppings are chosen, the program will add the topping onto the pizza, and ask the user if they would like cheese or pepper.

At the end of the game, the program will display a delivery image.
